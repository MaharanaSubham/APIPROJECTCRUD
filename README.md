This is a backend-only project built with ASP.NET Core Web API, designed to demonstrate basic API functionality using an in-memory database.

  Key Features:
- Framework: ASP.NET Core Web API
- Database: In-memory storage (no external DB required)
- Architecture: Backend-only, focused on API endpoints
- Data Flow: Communicates via HTTP endpoints (GET, POST, PUT, DELETE)
- Deployment: Live and Hosted in https://render.com/
- Purpose: Ideal for learning, prototyping, and testing API logic without persistent storage
